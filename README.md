# AI Web Explorer

##  簡介
AI Web Explorer 是一個結合 Google ADK（Agent Development Kit）與大型語言模型（LLM）的智慧搜尋助理。  
它能即時搜尋網路資訊，並由 AI 自動整理成清晰、客觀的回答。

此專案示範如何使用 `google.adk` 套件建立具搜尋能力的 AI Agent，並顯示：
1. 原始搜尋結果（Raw Search Results）
2. 經 AI 整理後的摘要回答（Summarized Response）
